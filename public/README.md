# Signals

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 20.3.7.

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.

## How to Use This Tutorial
- Start with SIGNALS-DEEP-DIVE.md - Read the overview and core concepts
- Explore signal-patterns-demo.component.ts - [See practical patterns in action](http://localhost:4200/signals-patterns)
- Study signal-patterns.ts - [Learn reusable helper functions](src\app\utils\signal-patterns.ts)
- Deep dive into signal-advanced.component.ts - [Master advanced techniques](http://localhost:4200/signals-advanced)
- Check signal-internals-demo.component.ts - [Interactive demos of internal concepts](http://localhost:4200/signals-internals)
- Read the documentation - [View markdown docs in-browser](http://localhost:4200/signals-deep-dive)
- Reference signal-internals.ts - [Understand performance and internals](src\app\utils\signal-internals.ts)
